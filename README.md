# Банковский виджет операций

Проект предназначен для обработки и отображения банковских операций клиента.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-логин/ваш-репозиторий.git
   
2. Установите зависимости:
   ```bash
   poetry install
   
## Использование

### Модуль masks

Содержит функции для маскировки номеров карт и счетов.

### Модуль widget

Содержит функции для форматирования данных.

### Модуль processing

Содержит функции для фильтрации и сортировки операций.
