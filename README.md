# Widget

Учебный проект, демонстрирующий работу с банковскими транзакциями. Программа умеет:

- загружать операции из файлов формата JSON, CSV и (при наличии `openpyxl`) XLSX;
- маскировать номера счетов и карт;
- искать транзакции по ключевым словам в описании с помощью регулярных выражений;
- подсчитывать количество операций по выбранным категориям описаний;
- интерактивно фильтровать, сортировать и выводить транзакции в консоль.

## Быстрый старт

```bash
poetry install  # при необходимости установить зависимости для разработки
python main.py
```

Следуйте подсказкам в консоли: выберите источник данных, статус операций и дополнительные параметры фильтрации. По умолчанию данные берутся из файлов `data/operations.json` и `data/operations.csv`.

## Тестирование и статический анализ

```bash
pytest
flake8
isort --check-only .
mypy .
```

`pytest` покрывает ключевые функции проекта, включая регулярные выражения, подсчёт категорий и консольную логику.
