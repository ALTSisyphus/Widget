# Банковский виджет операций

Проект предназначен для обработки и отображения банковских операций клиента.

## Установка

1. Клонируйте репозиторий с помощью веб-URL:
   ```bash
   git clone https://github.com/ALTSisyphus/Widget.git
   
2. Установите зависимости:
   ```bash
   poetry install
   
## Использование

### Модуль masks

Содержит функции для маскировки номеров карт и счетов.

### Модуль widget

Содержит функции для форматирования данных.

### Модуль processing

Содержит функции для фильтрации и сортировки операций.

## Тестирование
1. Установить библиотеку ``pytest-cov`` (плагин для фреймворка pytest)
   ```bash
   poetry add --group dev pytest-cov
2. Команды, чтобы запустить тесты с оценкой покрытия:
   ```bash
   pytest --cov``# При активированном виртуальном окружении
   poetry run pytest --cov # Через poetry
